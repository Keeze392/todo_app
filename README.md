### **About**
A simple ToDo App, run GUI using egui.

### **How to use**
You can download by [Release](https://github.com/Keeze392/todo_app/releases/tag/0.1.0).

To run: \
Simply click execute file to run app.

### **Build**
If you wish build outside of Linux platform, you need rust, cargo and such.
1. Get clone from this repo: \
`git clone https://github.com/Keeze392/todo_app.git`

2. Enter inside todo_app Directory then run with cargo: \
`cargo run --release`

It should open GUI, Release file is in target/release/ path.

Enjoy.
